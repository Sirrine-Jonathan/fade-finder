# Fade Finder — Local Barbers. On Demand.

![Fade Finder Logo](public/logo.png)

## Overview
Fade Finder is an on-demand marketplace connecting clients with local barbers offering both mobile/house-call appointments and studio/storefront services.

### App Concept & Features
- **User Types**:
  - Barbers looking to book clients (In-Studio & House Call)
  - Clients looking for barbers in their area
- **Core Features**:
  - GPS & Zip Code location search with distance filtering
  - Custom barber profiles with portfolio gallery & DOPL license checks
  - Dual-tier pricing (In-Studio rate vs. House Call rate)
  - Availability calendar & customizable booking approvals
  - Bi-directional client & barber reviews
  - In-app appointment reminders & notifications
